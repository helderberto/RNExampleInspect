## How to run locally

- Navigate to the project folder: `cd ./RNExampleInspect`;
- Install project dependencies: yarn;
- Install CocoaPods iOS packages: cd ios && pod install;
- Run in iOS Simulator: yarn ios;
- Run in Android Simulator: yarn android;
